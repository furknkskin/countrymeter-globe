# CountryMeter Globe

Bu klasör CountryMeter için ilk CesiumJS 3D dünya prototipidir.

## GitHub Pages kurulumu

1. GitHub'da `countrymeter-globe` isimli public repository oluştur.
2. Bu klasördeki `index.html` dosyasını repository kök dizinine yükle.
3. Repository içinde Settings > Pages bölümüne gir.
4. Source olarak `Deploy from a branch` seç.
5. Branch: `main`, Folder: `/ (root)` seçip Save'e bas.
6. Yayın adresini FlutterFlow WebView URL alanına gir.

Bu ilk sürüm:
- mouse ve dokunmatik ile döndürme,
- yakınlaştırma,
- otomatik yavaş dönüş,
- mobil ve masaüstü tarayıcı desteği

sağlar.
